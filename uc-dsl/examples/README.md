UC DSL Examples
====================================================================

This directory contains examples using the UC DSL.

* [`smc2`](smc2) is a case study consisting of two way secure message
  communication (SMC2), built from one way SMC, which is in turn built
  from Diffie-Hellman key exchange and one-time pad encryption.

* [`reentrancy`](reentrancy) is an example involving reentrancy,
  where the adversary gives control to the environment which
  reenters the real functionality.
