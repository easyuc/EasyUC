EasyUC Scripts
================================================================================

The bash shell script [`easyuc-merge`](easyuc-merge) is used to merge
a backed up theory generated by the translator with a newly generated
theory. See the comments in this file for usage information.

The perl script [`easyuc-cloc`](easyuc-cloc) is an adaptation of the
perl script `cloc` for counting non-commented, non-blank lines of code
to also include `.ec`, `.eca` and `.uc`, files, treating them the same
as OCaml code (as they share the same comment syntax).

The script [`easyuc-scripts-install`](easyuc-scripts-install) installs
these scripts as symbolic links in `../bin`, creating this directory
if necessary.
